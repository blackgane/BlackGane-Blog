## Java生态技术体系科普

#### Java技术体系至少包括四个部分

1. Java设计语言
2. 各种平台上的JAVA虚拟机（JVM）
3. Java API类库
4. 一系列辅助工具，如javac

    1+2+3+4 = JDK

    2+3 = JRE （Java运行环境）

    JDK > JRE > JVM
